# MarketFlow — MERN E-Commerce Platform

MarketFlow is a full-stack MERN e-commerce platform with customer shopping features, admin product/order management, secure authentication, payments, file uploads, analytics, and AI-powered enhancements.

## Current Status

- Stage 1 completed: Project requirements locked.
- Stage 2 completed: Database design completed.
- Stage 3 completed: Backend Express setup using ES module import/export style.
- Stage 4 completed: MongoDB connection and base error handling.
- Stage 5 completed: Base Mongoose ecommerce models.
- Stage 6 completed: Category CRUD APIs with validation.
- Stage 7 completed: Product create API with category relationship validation.
- Stage 8 completed: Product read APIs with category population.
- Stage 9 completed: Product update and delete APIs with validation.
- Stage 10 completed: Product search, filter, sort, and pagination.
- Stage 11 completed: Product indexes and query performance basics.
- Stage 12 completed: Category query features and safe category deletion.
- Stage 13 completed: User model preparation and auth planning.

## Tech Stack

- MongoDB
- Express.js
- React
- Node.js
- Mongoose
- Tailwind CSS
- JWT Authentication
- Admin Dashboard
- AI Integration
