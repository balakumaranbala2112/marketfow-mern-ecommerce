# MarketFlow — MERN E-Commerce Platform

## Project Goal

Build a full-stack e-commerce platform where customers can browse products, search/filter items, manage cart, place orders, make payments, and admins can manage products, categories, orders, users, analytics, and AI-powered product tools.

## User Roles

### Customer

- Register and login
- Browse products
- Search, filter, sort, and paginate products
- View product details
- Add products to cart
- Manage cart quantity
- Place orders
- View order history
- Add reviews and ratings
- Manage profile
- Reset password

### Admin

- Manage products
- Manage categories
- Manage orders
- Manage users
- View dashboard analytics
- Upload product images
- Manage coupons
- Use AI product tools

## MVP Features

- Authentication
